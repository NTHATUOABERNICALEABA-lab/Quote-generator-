# Quote Generator

**Description:**  
A motivational quote generator that displays random inspirational quotes with a click. Perfect for daily encouragement and keeping users motivated with fresh quotes every time.

**Features:**
- Generate random motivational quotes
- Minimal, clean design
- Easy-to-use interface

**How to Use:**
1. Open the page.
2. Click "New Quote" to get an inspirational message.

**Technologies Used:**
- HTML
- CSS
- JavaScript
